この拡張プラグインをご利用になるには、以下の手順にて作業してください。


◆【初心者向け】WordPress管理画面からインストールする場合 :
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 「digipress-ex-popular-posts.zip」をデスクトップなどにダウンロードします。
   ※圧縮ファイルは解凍する必要はありません。

2. WordPress管理画面の「プラグイン」メニュー内の「新規追加」サブメニューを
   クリック。

3.「アップロード」を選択。

4. "zip形式のプラグインをインストール" にて「ファイルを選択」ボタンをクリック。

5. "digipress-ex-popular-posts.zip" ファイルを選択して「いますぐインストール」
   　をクリック。

6. プラグインを正常に利用するには以下の手順にてライセンスの認証を行ってください。
   ●ライセンス認証手順 : 
   　https://digipress.digi-state.com/manual/how-to-download-and-activate/


◆FTPでアップロードする場合 :
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. ダウンロードした「digipress-ex-popular-posts.zip」を解凍します。
   ※「digipress-ex-popular-posts」というフォルダが生成されます。

2. このREADMEファイルがあるフォルダ(digipress-ex-popular-posts)を
   ご利用のWordPressがインストールされているサーバーのプラグインフォルダに
   FTPクライアントでアップロードします。
   ※例： /public_html/home/myname/wp-content/plugins

   ●おすすめFTPクライアント:FileZilla
    http://sourceforge.jp/projects/filezilla/

3. WordPressのプラグイン管理画面から「DigiPress Ex - Popular Posts」
   プラグインを有効化してください。

4. プラグインを正常に利用するには以下の手順にてライセンスの認証を行ってください。
   ●ライセンス認証手順 : 
   　https://digipress.digi-state.com/manual/how-to-download-and-activate/

5. 投稿画面やテキストウィジェットにてショートコードを利用します。


各ショートコードの詳細や使い方は以下のユーザーズサイトのオンラインマニュアル
をご覧ください。


-------------------------------------------------------------
◆DigiPressショートコード一覧／使い方：
-------------------------------------------------------------
 http://digipress.digi-state.com/manual/shortcode-list/



■注意事項：
 ・このプラグインはDigiPressテーマ専用のため、他のテーマでは正常に動作しません。
 ・ご利用のテーマのバージョンが古い場合やプラグインに未対応のテーマでは
   ご利用できません。
